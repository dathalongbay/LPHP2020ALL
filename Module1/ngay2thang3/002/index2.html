thuộc tính position trong css
        các giá trị của position 
        position: static; ( giá trị mặc định ) có nghĩa là tĩnh
        position: relative; ( tương đổi )
        position: absolute; ( tuyệt đối )
        position: fixed; ( cố định )
        position: sticky; ( con lai fixed và relative )


        4 tọa độ trên trình duyệt :
        top ( bên trên )
        left ( trái )
        right ( phải )
        bottom ( bên dưới )
        chú ý : top, left, right, bottom chỉ có tác dụng khi position khác static 


        chú ý thuộc tính position là relative thì nó sẽ dịch chuyển so với vị trí ban đầu của nó
        hay nói cách khác nó sẽ lấy vị trí ban đầu làm gốc tạo độ để dịch chuyển

Thực hành position: relative; có nghĩa là tương đối
