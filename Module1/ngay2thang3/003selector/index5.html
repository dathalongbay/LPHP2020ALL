SELECTOR mở rộng ( bộ chọn trong css )

        .classA : chọn tất cả thẻ có class css là classA 
        .classA.classB : chọn tất cả các thẻ có 2 class đồng thời là classA và classB 
        .classA .classB : chọn tất cả các thẻ có classB nằm trong classA 
        .classA > .classB : chọn tât cả các thẻ có classB là con trực tiếp của classA 
        .classA,.classB : chọn các thẻ có classA hoặc classB
