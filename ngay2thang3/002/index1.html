thuộc tính position trong css
        các giá trị của position 
        position: static; ( giá trị mặc định ) có nghĩa là tĩnh
        position: relative; ( tương đổi )
        position: absolute; ( tuyệt đối )
        position: fixed; ( cố định )
        position: sticky; ( con lai fixed và relative )

Thực hành position: relative; có nghĩa là tương đối
