Lưới ( grid ) trong bootstrap
        Mặc định bootstrap sử dụng hệ thống lưới 12 cột

        Làm quen với các class css của phần cột trong bootstrap

        trong 1 .container có thể chứa .row hàng
        trong 1 hàm .row có thể chứa nhiều cột

        class css .col mặc định tự động chia cột cho bootstrap
