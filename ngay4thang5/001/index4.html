class css .col chia giao diện 1 cách tự động
        làm quen với class css cho col nhưng chia theo độ rộng
        theo từng loại màn khác nhau
        responsive :
        đáp ứng theo các loại màn hình khác nhau
        ví dụ :
        giao diện mobile hiển thị 1 cột
        giao diện tab máy tính bảng hiển thị 2 cột
        giao diện laptop hiển thị 3 cột
        giao diện màn hình máy tính hiểu tính 4 cột

        .col- (extra small devices - screen width less than 576px)
        áp dụng mobile ( màn hình có độ rộng nhỏ 576px )

        .col-sm- (small devices - screen width equal to or greater than 576px)
        áp dụng cho màn hình máy tính bảng ( đô rộng bằng hoặc lớp hơn 576px - 767px )

        .col-md- (medium devices - screen width equal to or greater than 768px)
        áp dụng laptop ( width từ 768px đến 991px )

        .col-lg- (large devices - screen width equal to or greater than 992px)
        áp dụng màn hình pc ( độ rộng 992px đến 1199px )

        .col-xl- (xlarge devices - screen width equal to or greater than 1200px)
        áp dụng tivi ( màn hình width lớn hơn hay bằng 1200px )

        Kết luận : rút ra công thức sử dụng hậu tố
        col-sm-*
        * là 1 số hậu tố để chía cột
        1 cột => col-sm-12
        2 cột => col-sm-6
        3 cột => col-sm-4
        4 cột => col-sm-3

        12/số cột = hâu tố đằng sau class cột

        6 cột => 12/6 => col-sm-2
        12 cột => 12/12 => cok-sm-1
